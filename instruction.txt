команды для пользователя: 

rx [номер пина] [файл для вывода] - Установка считывания данных с пина в файл

tx [номер пина] [файл для ввода] - Установка ввода данных из файла в пин

com [файл с программой] - Установка файла с программой для мк

run - Запуск процесса



команды для программирования:

обозначения: r - адресс регистра(вначале пишется $), m - адресс памяти, p -номер пина

всего может быть не более 255 команд в файле прошивки

!!!

mov [r/m] [r/m] - перемещает в первую память значение из второго

add [r] [r/m] - добавляет к первому второе

sub [r] [r/m] - вычитает из первого второе

mul [r] [r/m] - умножает первое на второе

div [r] [r/m] - делит первое на второе

mod [r] [r/m] - остаток от деления первого на второе

and [r] [r/m] - логическое и 

or  [r] [r/m] - логическое или

xor [r] [r/m]

shl [r] [r/m] - побитовый сдвиг первого влево 

shr [r] [r/m] - побитовый сдвиг первого вправо

cmp [r] [r/m] - определяет значение логической переменной 1, если первое больше второго и тд

jg  [r/m] - если лог переменная = 1 то переходит в командах на строчку значения

jge [r/m] - если лог переменная = 1\0 то переходит в командах на строчку значения

jl  [r/m] - если лог переменная = -1 то переходит в командах на строчку значения

jle [r/m] - если лог переменная = -1\0 то переходит в командах на строчку значения

jmp [r/m] - переходит в командах на строчку значения

get [r] [p] - делает последний бит регистра равный значению пина

set [r] [p] - посылает в пин значение последнего бита регистра
